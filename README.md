# static-web-coins


Write a program that will convert a dollar amount to the number of coins that make up the
amount. Use the common American coins of quarters, dimes, nickels, and pennies.

```js

example:

Input: 0.67
  Output:
  {
    quarters: 2,
    dimes: 1,
    nickels: 1,
    pennies: 2
}

```
